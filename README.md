# Microsoft 365 / Entra ID Admin Lab

## Overview
A cloud administration lab using a Microsoft 365 Business Premium trial tenant to simulate real-world IT support and cloud admin workflows.

## What Was Configured
- Microsoft 365 tenant setup with user accounts and Business Premium license assignment
- Microsoft 365 groups and Teams for department-based collaboration
- Password resets, shared mailbox access, and account offboarding
- Microsoft Entra Security Defaults — baseline MFA protection verified
- Real helpdesk tickets worked in osTicket — onboarding, shared mailbox access, cloud offboarding
- Knowledge base articles documenting repeatable M365 admin procedures

## Tools Used
- Microsoft 365 Business Premium
- Microsoft 365 Admin Center
- Microsoft Entra Admin Center
- Microsoft Teams Admin Center
- osTicket
